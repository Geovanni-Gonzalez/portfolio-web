Sobre mí

Estudiante de Ingeniería en Computación en el Tecnológico de Costa Rica, con enfoque en el desarrollo de software y la resolución de problemas. Me apasiona la investigación y el aprendizaje continuo, entender cómo funcionan las tecnologías y cómo aplicarlas para crear soluciones innovadoras.

He participado en el desarrollo de proyectos que van desde aplicaciones web interactivas hasta sistemas de gestión de datos distribuidos. Mi objetivo es combinar buenas prácticas de desarrollo con un diseño centrado en el usuario para construir software de calidad.

CalculadoraMultibase-DesktopApp

Calculadora multibase desarrollada con JavaScript y Electron. Permite realizar conversiones entre diferentes bases numéricas y cuenta con una interfaz gráfica intuitiva.

JavaScriptCSSNode.jsElectronHTML

Match-3

Juego multijugador competitivo tipo Match-3 desarrollado con stack moderno (React 19 + Node.js). Utiliza WebSockets para sincronización en tiempo real, permitiendo partidas simultáneas con validación de movimientos en el servidor y persistencia de puntajes en MySQL.

ReactNode.jsTypeScriptSocket.IOMySQL
Github

Aventura-del-Tesoro-Perdido

Juego de aventura híbrido que integra una interfaz gráfica moderna en WPF (C#) con un motor de lógica experto en Prolog. Cuenta con sistema de inventario, resolución de acertijos y navegación inteligente gestionada por reglas lógicas.

C#WPFProlog.NETXAML
Github

Ahorcado-ConsolApp

Aplicación de consola en Python para el juego del Ahorcado. Implementado desde cero con restricciones técnicas estrictas (sin funciones built-in básicas), incluye gestión administrativa, modos de juego, soporte bilingüe e innovaciones visuales y de jugabilidad.

Python
Github

Ahorcado-GUI-App

Aplicación de escritorio moderna del juego Ahorcado. Incluye una interfaz gráfica pulida, sistema de gestión de palabras, y modos de juego innovadores como Duelo y Blitz. Destaca por su implementación algorítmica desde cero sin funciones predefinidas de Python.

PythonTkinter
Github

Ahorcado-GUI-App-POO

Versión orientada a objetos del juego Ahorcado. Implementa una arquitectura robusta separando lógica y presentación. Incluye características avanzadas como Duelo Local, Modo Blitz, y un sistema administrativo seguro, cumpliendo estrictos estándares de codificación sin funciones predefinidas.

PythonTkinter
Github

BattleshipARM

Implementación de bajo nivel del juego Batalla Naval en Ensamblador ARM. Ejecutado en QEMU/Linux, destaca por su gestión directa de registros, manejo seguro de la pila, lógica de IA con RNG basado en registros y una interfaz visual pulida con códigos ANSI.

ARM AssemblyGNU ToolchainMakeShell Scripting
